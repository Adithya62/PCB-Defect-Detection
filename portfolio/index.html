Project Overview

This project focuses on semantic-aware PCB defect detection using state-of-the-art deep learning techniques, including YOLO and CBAM attention mechanisms. The goal is to improve the detection of tiny defects on printed circuit boards by leveraging both classification and object detection approaches.

Toggle

Week 4: Binary Classifier Development

Toggle
  • Created binary dataset merging 6 defect classes into single "defective" class
  • Balanced with normal patches (4 per temp image, total 6,000 normal patches)
  • Total: 10,013 defect + 6,000 normal patches
  • Used VGG16 with binary output (sigmoid activation)
  • Applied class weight: defect=5, normal=1 to boost recall
  • Overall accuracy: 82%
  • Defect recall: 0.63, defect precision: 0.85
  • Normal recall: 0.93, normal precision: 0.81
  • Identified critical issue: 37% of defects missed (false negatives)
  • Determined binary classifier not suitable for production

🔑 Key Findings

  1. VGG16 patch classifier achieved 98% accuracy but required pre-cropped patches
  2. Binary classification at patch-level fundamentally limited (37% false negatives)
  3. CBAM attention made model more conservative, requiring threshold re-tuning
  4. YOLO with full images outperforms patch-based classifiers (96% mAP)
  5. Preprocessing (CLAHE + Top-Hat) improves mAP by 2%
  6. YOLO is production-ready: 140 FPS + bounding boxes

📝 Lessons Learned

  • Always start with simple baseline before complex models
  • Full images provide better context than isolated patches
  • YOLO is optimal balance of speed and accuracy for PCB inspection
  • Threshold tuning critical when using attention modules
  • Document everything weekly - saves time during final reporting
